# QA Automation Lab Web

Laboratorio práctico de desarrollo Frontend y QA Automation construido con React y TypeScript.

El proyecto tiene como objetivo servir como entorno de aprendizaje para automatización de pruebas web mediante herramientas utilizadas en la industria, manteniendo una arquitectura profesional y un enfoque orientado a la calidad desde el inicio del desarrollo.

---

## Objetivos

- Aprender React y TypeScript.
- Consumir APIs REST desarrolladas en el laboratorio Backend.
- Implementar autenticación basada en JWT.
- Automatizar pruebas End-to-End.
- Aplicar buenas prácticas de arquitectura Frontend.
- Construir un portafolio orientado a QA Automation.

## Ejecución Local

### Instalar dependencias

```bash
npm install
```

### Ejecutar la aplicación

```bash
npm run dev
```

### Ejecutar ESLint

```bash
npm run lint
```

### Verificar formato

```bash
npm run format:check
```

### Formatear el proyecto

```bash
npm run format
```

### Generar Build

```bash
npm run build
```

## Filosofía

Este proyecto no pretende construir la aplicación visualmente más compleja.

El objetivo es desarrollar una aplicación suficientemente realista para practicar desarrollo Frontend y automatización de pruebas utilizando herramientas empleadas en entornos profesionales.

Cada Sprint incorporará nuevas funcionalidades junto con su correspondiente estrategia de pruebas automatizadas.
